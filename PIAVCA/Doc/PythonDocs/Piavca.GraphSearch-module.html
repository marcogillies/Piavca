Function Details [hide private]

search(G, search_strategy, initial_vertex=whole_graph)

source code 

Generate sequence of triples (v,w,edgetype) for DFS of graph G. The subsequence for each root of each tree in the DFS forest starts with (root,root,forward) and ends with (root,root,reverse). If the initial vertex is given, it is used as the root and vertices not reachable from it are not searched.
