Piavca can be downloaded from sourceforge. http://sourceforge.net/projects/piavca/

Installation

Python Version

You will need to install all prerequisites first (see below)

If you are running windows and Python 2.6 there is a binary installer, just run that.

For Mac and linux you need to use the standard python setup.

You will need to use a terminal

If you have root access (generally this means that you own the machine), type the following:

python setup.py build
python setup.py install
If it complains that you don't have permissions try: 
python setup.py build
sudo python setup.py install
You will need to type in your password. If you still haven't got permissions try installing somewhere else:

python setup.py build
python setup.py install -prefix
It might also fail because you haven't got a compiler (gcc) installed or the appropriate libraries, make sure you get the prerequisites

Prerequisites

Unless you are using a windows binary installer, you will need a compiler.

For linux make sure you have the gcc package installed and the standard header files.

For a mac install the development tools (xcode)

http://developer.apple.com/mac/

For windows you can install various compilers, Microsoft Visual C++ is an obvious free (beer) version:

http://www.microsoft.com/exPress/

Python Version

To run the python version you will need:

Python:

http://www.python.org/

wxPython (a gui toolkit)

http://www.wxpython.org/

PyOpenGL (access to the OpenGL 3D library)

http://pyopengl.sourceforge.net/

For some functions you might need scipy maths libraries (but unlikely)

http://www.scipy.org/
