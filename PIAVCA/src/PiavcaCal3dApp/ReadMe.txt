========================================================================
    PiavcaCal3dApp Project Overview
========================================================================

This is an example application using only the C++ interface.

You pass the name of a cal3d configuration file (.cfg, see the cal3d docs of details) 
and it will display the character. pressing the number keys will play different animations.

See the main function for details of how to set up a Glut application with Piavca and cal3d.
The displayFunc function for how to update and render piavca avatars and the play_motion function
for how to play an animation
